# SB-Shop

SB-Shop is a microservices-based online store built using Java 21, Spring Boot 3 and Angular 19.

## Technologies Used

* Java 21
* Spring Boot 3
* Angular 19
* MySQL 8
* Apache Tomcat
* MongoDB
* Docker
* Testcontainers
* RestAssured
* WireMock
* Git
* IntelliJ IDEA
* VS Code
* Postman
